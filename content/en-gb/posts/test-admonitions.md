# Testing Admonitions

This post tests the hugo-admonitions module with GitHub flavored markdown syntax.

> [!NOTE]
> This is a note admonition. Use it to highlight important information.

> [!TIP]
> This is a tip admonition. Use it to provide helpful advice.

> [!IMPORTANT]
> This is an important admonition. Use it for critical information.

> [!WARNING]
> This is a warning admonition. Use it to alert users about potential issues.

> [!CAUTION]
> This is a caution admonition. Use it for dangerous actions or situations.

## Custom Titles

You can also use custom titles:

> [!NOTE] Custom Note Title
> This note has a custom title!

> [!TIP] Pro Tip
> Here's a professional tip with a custom title.
